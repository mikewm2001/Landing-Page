Projects

I proudly present what I have been doing.

Below you will find several projects that showcase different skills that I have gained throughout the past years. I firstly list an About-Me project where I attempted to build my first personal website in HTML, CSS, and JavaScript. You can learn even more about me through there.

Secondly I have built my first full-stack application in React with a Flask framework in the backend. This project was sparked by the thought of inefficiency in memorizing hundreds of different passwords. Since I did not want to use notepad or Google Sheets to manage my credentials and I wanted to improve my programming skills, I decided to make a web application to remediate this issue.

Thirdly, I have built a project with a teammate involving Android SDK, Google Material Design, Firebase, and Kotlin. The goal of this project was to combine the best features of different nutrition and strength tracking mobile applications into one.

Lastly, in a team of 3, I am collaborating to build a full-stack web application to greatly decrease the time spent in managing job applications. I know that I, for example, spend too much time recording the details of a job application on a Google Sheet. To resolve this issue we decided to automate this logging on a web application. This project is being built with Amazon RDS, PostgreSQL, Node, Express, and WebDriverIO.
